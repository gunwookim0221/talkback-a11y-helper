# UI 캡처 데이터셋

## 목적
이 폴더는 SmartThings 앱에서 추출한 스크린샷, XML, 메타데이터를 모아둔 참조 데이터입니다.

주요 목적은 다음과 같습니다.
- TalkBack 포커스 이동 분석
- SMART_NEXT 동작 검토 및 수정
- 상단바 / 콘텐츠 / 하단바 구조 분류
- 스크롤 이후 continuation 처리 검토

## 폴더 구성 원칙
- 최상위 폴더는 주요 탭 또는 화면 그룹 단위로 나눕니다.
- 하위 폴더는 화면 상태 또는 스크롤 단계 단위로 나눕니다.
- 각 step은 필요 시 아래 파일을 가질 수 있습니다.
  - `devices_step_XX.png`
  - `devices_step_XX.xml`
  - `devices_step_XX.meta.json`
  - `step_XX.md`

## 구현 참고 원칙
- 이 데이터는 구조 참고용이며, 특정 label 하드코딩용이 아닙니다.
- 아래와 같은 범용 규칙으로 처리해야 합니다.
  - 상단 app bar 분류
  - 콘텐츠 영역 경계 판별
  - 하단 bottom navigation 제외
  - 스크롤 후 새로 나타난 후보 우선 처리
- `Labs`, `SmartThings`, `Devices`, `Menu` 같은 특정 문자열에 의존해 예외 처리하지 않습니다.

## 참고 사항
- 일부 탭은 한 화면 안에 전체 구조가 보입니다.
- 일부 탭은 길어서 스크롤 후 여러 step으로 나누어 캡처했습니다.
- Settings 계열 화면은 하단 탭 루트와 성격이 다르므로 별도 폴더로 분리합니다.
